# GigaChat-CLI

Небольшое CLI‑приложение на Swift для общения с GigaChat через API.

## Требования

- macOS
- Xcode (для сборки) или Swift toolchain

## Запуск

Откройте проект:

- `GigaChat-CLI.xcworkspace` (если используете CocoaPods)
- или `GigaChat-CLI.xcodeproj`

Соберите и запустите таргет `GigaChat-CLI`.

## Настройка доступа

Сейчас ключи и базовые URL настраиваются в `GigaChat-CLI/Source/Chat/APIConstants.swift`.

Важно: не коммитьте реальные ключи в публичный репозиторий.
